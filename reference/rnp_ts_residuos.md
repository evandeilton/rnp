# Diagnostico de residuos de modelo de series

Ljung-Box e normalidade dos residuos de um ajuste ARIMA/SARIMA.

## Usage

``` r
rnp_ts_residuos(modelo, lag = 10, digits = 4L)
```

## Arguments

- modelo:

  Saida de
  [`rnp_arima()`](https://evandeilton.github.io/rnp/reference/rnp_arima.md)/[`rnp_sarima()`](https://evandeilton.github.io/rnp/reference/rnp_sarima.md)
  (ou objeto `Arima`).

- lag:

  Defasagens para o Ljung-Box.

- digits:

  Inteiro.

## Value

tibble com os testes de diagnostico.
