# Fator de inflacao de variancia (VIF)

Mede a multicolinearidade: VIF_j = 1 / (1 - R_j^2), onde R_j^2 e o
R-quadrado da regressao do preditor j sobre os demais.

## Usage

``` r
rnp_vif(modelo, digits = 4L)
```

## Arguments

- modelo:

  Objeto `lm`.

- digits:

  Inteiro.

## Value

tibble com `termo`, `vif` e `interpretacao`.

