# Bootstrap (nao parametrico)

Gera `B` replicas bootstrap de uma estatistica e estima vies e
erro-padrao. Estatisticas comuns (`"media"`, `"mediana"`, `"dp"`,
`"variancia"`) usam backend C++; uma funcao R arbitraria tambem e
aceita.

## Usage

``` r
rnp_bootstrap(
  x,
  estatistica = "media",
  B = 2000L,
  na.rm = TRUE,
  seed = 42L,
  digits = 4L
)
```

## Arguments

- x:

  Vetor numerico.

- estatistica:

  String (`"media"`, `"mediana"`, `"dp"`, `"variancia"`) ou funcao
  `function(x) ...`.

- B:

  Inteiro. Numero de replicas.

- na.rm:

  Logico.

- seed:

  Inteiro. Semente.

- digits:

  Inteiro.

## Value

Uma lista com `replicas` (vetor) e `resumo` (tibble com `estimativa`,
`media_boot`, `vies`, `erro_padrao`, `B`).

## Examples

``` r
set.seed(1)
rnp_bootstrap(rnorm(100), "media", B = 1000)$resumo
#> # A tibble: 1 × 5
#>   estimativa media_boot    vies erro_padrao     B
#>        <dbl>      <dbl>   <dbl>       <dbl> <dbl>
#> 1      0.109      0.107 -0.0019      0.0901  1000
```
