# Grafico de serie temporal

Grafico de serie temporal

## Usage

``` r
rnp_grafico_serie(x, titulo = NULL)
```

## Arguments

- x:

  Vetor numerico ou `ts`.

- titulo:

  Titulo opcional.

## Value

Objeto `ggplot`.
