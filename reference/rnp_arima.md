# Ajuste de modelo ARIMA

Ajusta um modelo ARIMA(p, d, q) por maxima verossimilhanca
([`stats::arima()`](https://rdrr.io/r/stats/arima.html)), com saida
tidy.

## Usage

``` r
rnp_arima(x, ordem = c(0, 0, 0), incluir_media = TRUE, digits = 4L)
```

## Arguments

- x:

  Vetor numerico ou `ts`.

- ordem:

  Vetor `c(p, d, q)`.

- incluir_media:

  Logico. Inclui media/intercepto (apenas se `d = 0`).

- digits:

  Inteiro.

## Value

Uma lista com `coeficientes`, `modelo` (AIC/BIC/AICc/sigma2), `objeto`
(`Arima`) e `serie` (a serie original).
