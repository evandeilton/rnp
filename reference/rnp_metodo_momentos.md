# Estimacao por metodo dos momentos

Estima parametros de distribuicoes comuns igualando momentos amostrais
aos teoricos.

## Usage

``` r
rnp_metodo_momentos(
  x,
  dist = c("norm", "exp", "pois", "gamma", "beta", "unif"),
  digits = 4L
)
```

## Arguments

- x:

  Vetor numerico.

- dist:

  String: `"norm"`, `"exp"`, `"pois"`, `"gamma"`, `"beta"`, `"unif"`.

- digits:

  Inteiro.

## Value

tibble com `parametro` e `estimativa`.
