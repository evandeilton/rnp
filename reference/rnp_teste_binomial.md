# Teste binomial exato

Wrapper de
[`stats::binom.test()`](https://rdrr.io/r/stats/binom.test.html).

## Usage

``` r
rnp_teste_binomial(
  sucessos,
  n,
  p0 = 0.5,
  lado = c("bilateral", "direita", "esquerda"),
  digits = 4L
)
```

## Arguments

- sucessos:

  Numero de sucessos.

- n:

  Tamanho da amostra.

- p0:

  Proporcao sob H0.

- lado:

  String: `"bilateral"`, `"direita"`, `"esquerda"`.

- digits:

  Inteiro.

## Value

tibble com `proporcao`, `p_valor`, `ic_inf`, `ic_sup`, `alternativa`.
