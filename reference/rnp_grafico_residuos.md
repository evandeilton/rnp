# Painel de diagnostico de residuos

Gera os graficos classicos de diagnostico de um `lm`.

## Usage

``` r
rnp_grafico_residuos(modelo)
```

## Arguments

- modelo:

  Objeto `lm`.

## Value

Uma lista nomeada de objetos `ggplot`: `residuo_ajustado`, `qq`,
`escala_locacao`, `residuo_leverage`.
