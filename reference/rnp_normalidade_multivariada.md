# Teste de normalidade multivariada (Mardia)

Avalia assimetria e curtose multivariadas (teste de Mardia).

## Usage

``` r
rnp_normalidade_multivariada(X, digits = 4L)
```

## Arguments

- X:

  Matriz/data.frame numerica.

- digits:

  Inteiro.

## Value

tibble com `medida`, `estatistica`, `p_valor`.
