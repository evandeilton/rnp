# Decomposicao de serie temporal

Wrapper de
[`stats::decompose()`](https://rdrr.io/r/stats/decompose.html) e
[`stats::stl()`](https://rdrr.io/r/stats/stl.html).

## Usage

``` r
rnp_ts_decomposicao(
  x,
  type = c("classica", "stl"),
  frequency = NULL,
  digits = 4L
)
```

## Arguments

- x:

  Vetor numerico ou objeto `ts`.

- type:

  String: `"classica"` ou `"stl"`.

- frequency:

  Inteiro. Periodicidade (default `NULL` = auto).

- digits:

  Inteiro.

## Value

Uma lista com `serie` (objeto ajustado) e `componentes` (tibble).
