# Teste de proporcoes (uma ou k amostras)

Wrapper de
[`stats::prop.test()`](https://rdrr.io/r/stats/prop.test.html).

## Usage

``` r
rnp_teste_proporcoes(sucessos, n, p0 = NULL, corrigir = TRUE, digits = 4L)
```

## Arguments

- sucessos:

  Vetor de sucessos.

- n:

  Vetor de tamanhos amostrais.

- p0:

  Proporcao sob H0 (apenas uma amostra). Default `NULL`.

- corrigir:

  Logico. Correcao de continuidade.

- digits:

  Inteiro.

## Value

tibble com `estatistica`, `gl`, `p_valor`, `proporcoes`.

``` r

#> # A tibble: 1 × 4
#>   estatistica    gl p_valor proporcoes

rnp_teste_proporcoes(45, 100, p0 = 0.5)

#>   estatistica    gl p_valor proporcoes
```
