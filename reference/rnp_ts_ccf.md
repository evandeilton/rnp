# Funcao de correlacao cruzada (CCF)

Funcao de correlacao cruzada (CCF)

## Usage

``` r
rnp_ts_ccf(x, y, lag_max = NULL, digits = 4L)
```

## Arguments

- x, y:

  Vetores numericos.

- lag_max:

  Numero maximo de defasagens.

- digits:

  Inteiro.

## Value

Uma lista com `tabela` (tibble) e `grafico` (`ggplot`).
