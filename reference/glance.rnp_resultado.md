# Resumo do ajuste (glance) de um resultado do rnp

Extrai a tabela de medidas-resumo do modelo de um objeto `rnp_resultado`
(uma linha), no formato esperado pelo `broom`/`tidymodels`.

## Usage

``` r
# S3 method for class 'rnp_resultado'
glance(x, ...)
```

## Arguments

- x:

  Objeto de classe `rnp_resultado`.

- ...:

  Ignorado.

## Value

Um [tibble](https://tibble.tidyverse.org/reference/tibble.html) de uma
linha.
